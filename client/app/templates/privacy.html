Privacy Policy

User Data

o2r stores following user data in its database:

  • ORCID id
  • username
  • user level
  • date of last login

Cookies

o2r uses a session cookie. This cookie stores a unique reference to a session (including auth details, if the user is autheticated via orcid).

Login

o2r uses OAUTH2 to provide a login via ORCID. More information about the ORCID privacy policy here.

ORCID is known to cause issues when using ghostery and other tacking control software. More information here.
