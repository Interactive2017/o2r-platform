Privacy Policy

User Data

o2r stores following user data in its database:

  • internal user id
  • orcid id
  • username
  • user level
  • date of last login

Cookies

o2r uses a session cookie. This cookie stores a unique reference to a session (including auth details, if the user is autheticated via orcid).

Login

o2r uses OAUTH2 to provide a login via orcid. More information about the orcid.org privacy policy here.
